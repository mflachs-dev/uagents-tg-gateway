# @agentlaunch/uagents-tg-gateway

An open-source JavaScript runtime utility (designed for Node.js environments) that seamlessly connects Fetch.ai agents to Telegram channels and secure downstream webhooks on the BNB Chain. 

This package provides lightweight, plug-and-play tracking infrastructure to monitor agent wallet balances, operational telemetry, and token lifecycle events on the Agent Launchpad.

---

## ⚡ Core Features

*   **BNB Chain State Polling:** Uses standard JSON-RPC network providers to read wallet metrics, track token graduation states, and flag critical transaction thresholds.
*   **HMAC Payload Verification:** Packages alert metrics into clean JSON shapes and appends a SHA-256 HMAC signature to ensure target endpoints can verify data authenticity.
*   **Multi-Channel Broadcasting:** Routes alert notification arrays concurrently to remote webhooks and direct Telegram chat feeds using the native Telegram Bot API.
*   **Fail-Safe Simulation Mode:** Includes an isolated execution flag (`--simulate`) allowing developers and grant managers to test the entire parsing engine locally without live mainnet variables.

## ⚙️ Configuration Profile

The application utilizes environment variables for complete state isolation. Create a `.env` file in your project root using the configuration layout below:

| Environment Variable | Type | Description | Example Value |
| :--- | :--- | :--- | :--- |
| `BSC_RPC_URL` | String | Target BNB Chain RPC execution node | `https://bsc-dataseed.binance.org/` |
| `TARGET_AGENT_WALLET` | Address | Checksummed wallet tracking target | `0x031b41e5...fa7f` |
| `MIN_BALANCE_THRESHOLD`| Float | Safety floor for asset monitoring triggers | `1.5` |
| `TELEGRAM_BOT_TOKEN` | String | Authorization key generated by BotFather | `123456:ABC-def...` |
| `TELEGRAM_CHAT_ID` | String | Targeted group or individual messaging ID | `-100XXXXXXXXXX` |
| `WEBHOOK_DESTINATION_URL`| String | Downstream API handling payload routing | `https://api.domain.com/v1` |
| `WEBHOOK_SECRET_HMAC` | String | Private key used for SHA-256 signature generation | `your_secret_key` |
